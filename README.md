# C_Projetos

Este projeto foi desenvolvido como um desafio do **CS50 (Harvard)**, integrando conceitos de lógica de programação em C com fundamentos de [Engenharia Elétrica / Robótica] e Projetos Pessoais.

## ⚡ Visão Geral

O objetivo destes projetos é demonstrar o uso de **C** para Automação Industrial, Robótica dentro do Spectro na Engenharia Elétrica.

### 🛠 Stack Tecnológica (Engenharia & Dev)

* **Linguagem:** C (padrão C11)
* **Ambiente:** Linux (Fedora/Terminal/LazyVim)
* **Compilador:** GCC com flags de otimização
* **Bibliotecas:** `stdio.h`, `math.h` (para cálculos trigonométricos de incidência solar)

---

## 🚀 Funcionalidades para Engenharia

* **Cálculo de Precisão:** Implementação de funções matemáticas para [ex: torque de motores / eficiência energética].
* **Interface via Terminal:** Entrada de dados via `argc` e `argv` para automação de testes.
* **Gestão de Memória:** Uso eficiente de recursos, essencial para sistemas embarcados e microcontroladores.
